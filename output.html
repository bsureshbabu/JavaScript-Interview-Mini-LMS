Question: Event loop ordering
// From the PDF
setTimeout(() => console.log(1), 0);

console.log(2);

new Promise(res => {
  console.log(3)
  res();
}).then(() => console.log(4));

console.log(5);
Sample output (expected)
Console output: 
3
2
5
4
1
Explanation (recreated from PDF): Promise executor runs immediately (prints 3), then synchronous logs 2 and 5; microtask `.then` runs before macrotask `setTimeout`.

Question: What is output of `x + y` where x and y are arrays?
const x = [1,2,3];
const y = [1,3,4];
console.log(x + y);
Sample output
"1,2,31,3,4"
Explanation: arrays are converted to strings via `toString()` then concatenated.

forEach with break — what's the output?
const nums = [1,2,3,4,5,6,7];
nums.forEach(n => {
  if (n % 2 === 0) {
    break; // invalid inside forEach
  }
  console.log(n);
});
Sample output / note
// This will throw a SyntaxError or not behave as intended.
// 'break' is not allowed inside forEach callback; use a for-loop instead.
Explanation: `break` only works in loops (for/while), not inside callbacks. See PDF. :contentReference[oaicite:10]{index=10}
